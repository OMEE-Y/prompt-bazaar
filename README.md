# Prompt Bazaar

![React](https://img.shields.io/badge/React-17.0.2-blue?logo=react&logoColor=white) ![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow?logo=javascript&logoColor=black)

**Prompt Bazaar** is a React project built to practice the fundamentals of React while creating a practical tool.  

It’s a **marketplace for AI prompts**, designed to help you get better results from AI tools like ChatGPT or Claude.

---

## Features

- Browse a collection of **ready-to-use AI prompts**.  
- **Copy prompts** directly to your clipboard for instant use.  
- **Open prompts in AI tools** (like Claude or ChatGPT) with one click.  
- Add prompts to ChatGPT’s **customization tab** to make it behave like a tech expert.  
- Lightweight, fast, and fully responsive UI.
